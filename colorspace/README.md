The color axes in the human brain are R-G and B-Y. In color speak L = R, M = G, S = B (long, med, short wavelength).

See https://compcogneuro.org/vision#color for details.

* Here’s what is implemented: https://en.wikipedia.org/wiki/LMS_color_space – HPE Hunt-Pointer-Estevez, transforms RGB -> XYZ -> LMS -> color opponents which are the L, a, b values used in [CIECAM02](https://en.wikipedia.org/wiki/CIECAM02).

* Key paper: Moroney et al., 2002
